DIA Photometry and Gravitational Microlensing

In this project, I analysed gravitational microlensing events using difference image photometry techniques with data from the Angstrom Project. This study identified several microlensing events in the Andromeda Galaxy, including a brown dwarf and a stellar-mass black hole, as well as a nova. Key improvements were proposed, such as employing machine learning for parameter optimisation and using advanced models for more complex events. This work demonstrates the power of microlensing to reveal and characterise hidden astronomical objects.
